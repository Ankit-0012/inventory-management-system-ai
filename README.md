# Inventory Management System with AI Forecasting

This project is a web-based inventory management system developed using Flask, SQLite, and Machine Learning.

## Features
- User Login & Authentication
- Product Management
- Sales Tracking
- Analytics Dashboard
- AI-based Demand Forecasting

## Technologies Used
- Python (Flask)
- SQLite
- HTML, CSS, Bootstrap
- Machine Learning (Random Forest)

## How to Run
1. Download the ZIP file
2. Extract the project
3. Install required libraries
4. Run app.py

## Note
The complete project is available in ZIP format in this repository.
